🐧 Linux

Linux fundamentals for DevOps engineers. File system, permissions, processes, networking commands, shell scripting, and systemd service management.

chmod, chown, chgrp, SUID/SGID, sticky bit, ACLs explained.

Process Management

ps, top, htop, kill, nice, nohup, systemd, journalctl.

Shell Scripting (Bash)

Variables, conditionals, loops, functions, cron jobs, and script debugging.

Networking Commands

ip, ss, netstat, curl, wget, ping, traceroute, iptables basics.

Package Management

apt, yum/dnf, snap — install, update, remove, and repositories.

Storage & Disk

df, du, lsblk, fdisk, mount/umount, LVM basics.

SSH & Remote Access

SSH key auth, scp, rsync, port forwarding, ssh config file.
